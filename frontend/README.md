# Frontend

Carpeta para clientes web y mobile. Ambos consumen el backend via HTTPS y OIDC.

- web/: app web (SPA/SSR) con chat, panel y herramientas.
- mobile/: app mobile (React Native/Flutter). Enfasis en chat y notificaciones.
